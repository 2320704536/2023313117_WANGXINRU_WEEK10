# 🎬 Film Production Dashboard

A Streamlit web dashboard for analyzing film production data, including movie budgets, revenues, profits, genres, and ROI performance.

This project helps users understand how different films perform financially by comparing production costs, box office revenue, return on investment, and genre distribution through interactive charts and tables.

---

## 📌 Project Description

**Film Production Dashboard** is a data visualization web app built with Streamlit.

The dashboard allows users to explore sample film production data, add new films, filter and search movies, compare budgets and revenues, calculate ROI, and download the data as a CSV file.

It is designed for users who are interested in film, media production, box office performance, and data analytics.

---

## ✨ Features

- 📊 Production overview with key metrics
- 🎬 Add new film data through the sidebar
- 🔍 Search films by title or director
- 🎭 Filter films by genre
- 📋 Interactive film data table
- 💰 Budget vs revenue comparison chart
- 🚀 ROI analysis by film
- 🎭 Genre distribution chart
- 📅 Films by release year chart
- 🗑️ Delete films from the dataset
- 📤 Download film data as a CSV file
- 🎨 Clean, high-contrast dashboard design

## ▶️ How to Run

1. Clone this repository:

```bash
git clone [your-repository-link]
```

2. Move into the project folder:

```bash
cd [your-repository-name]
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the Streamlit app:

```bash
streamlit run app.py
```
